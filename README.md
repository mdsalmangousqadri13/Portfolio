- Responsive Design (Mobile Friendly)

## 💻 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Go to project folder:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
